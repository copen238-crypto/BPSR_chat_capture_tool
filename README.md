# BPSR_chat_tool（非公式・自己責任ツール）
スタレゾのチャットをOCR読取りして、文字起こしするツールです。

ゲーム画面のチャット欄を指定範囲でキャプチャ → OCR（Tesseract）で読み取り、別ウィンドウにログ表示するツールです。  
※運営・開発元とは一切関係ない **非公式ツール** です。

## 透明性
- このツールは **ネット通信しません**（外部送信なし・ネット不要）。
- ディスク書き込みは基本 **設定ファイル（json）** くらいです。
- OCRのために **tesseract.exe が外部プロセスとして起動**します（これは正常動作です）。
- 画面キャプチャは **指定した範囲のみ**を使います。

## インストール手順（必要なもの）

### 1) Tesseract（OCR本体）のインストール
UB Mannheim（ドイツ・マンハイム大学図書館）提供のインストーラーを使用します。

https://github.com/UB-Mannheim/tesseract/wiki

**※本ツールはC:\Program Files\Tesseract-OCR\tesseract.exeにインストールされる前提となっています。**

**※インストール時にadditional script dataとadditional language dataのjapaneseをチェックすること！**

### 2) Python のインストール（Microsoft Store）
Microsoct store

https://apps.microsoft.com/detail/9pnrbtzxmb4z?hl=ja-JP&gl=JP

### 3) 必要ライブラリのインストール
PowerShell（またはコマンドプロンプト）で以下を実行：

pip install pyautogui

pip install pytesseract

pip install opencv-python

## 起動方法
PowerShell（またはコマンドプロンプト）で以下を実行：

python BPSR_chat_tool.py
